# МЭШ — Автозаполнение Soft Skills ⚡

Автоматическое заполнение оценки «soft skills» для всех учеников в электронном журнале МЭШ (school.mos.ru) на основе их среднего балла по предмету.

---

## 🌟 Способ 1. Браузерное расширение (Рекомендуется для всех)
*Этот способ разработан специально для преподавателей. Не требует установки Python, работы с командной строкой и ручного копирования токенов из консоли разработчика. Работает на любой операционной системе (Windows, macOS, Linux, ChromeOS) в любом современном браузере.*

### 📋 Что умеет эта версия:
1. **Автоматическая авторизация**: Достаточно просто войти в свой аккаунт МЭШ в браузере.
2. **Красивый интерфейс**: Удобная панель управления прямо на странице электронного журнала.
3. **Выбор периода (триместра)**: Можно выбрать 1-й, 2-й, 3-й триместр или весь год. Скрипт сам рассчитает средний балл именно за выбранный период (решает проблему сбивания триместров в журнале).
4. **Интерактивное превью**: Перед отправкой вы увидите таблицу всех учеников с их средними баллами и рассчитанными ответами. Вы можете **вручную скорректировать любой ответ** для конкретного ученика прямо в таблице.
5. **Режим исправления**: Можно выбрать заполнение только новых (пропуская заполненных) или полную перезапись (удаление старых анкет и отправка новых).
6. **Настройка порогов без кода**: Регулируйте требования к оценкам с помощью простых ползунков в панели настроек.

---

### 🚀 Инструкция по установке (занимает 1 минуту):

#### Шаг 1. Установите расширение Tampermonkey
Tampermonkey — это безопасное и популярное расширение для запуска пользовательских скриптов.
* **Для Chrome / Яндекс.Браузер / Opera / Edge**: Установите из [Магазина Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
* **Для Firefox**: Установите из [Firefox Add-ons](https://addons.mozilla.org/ru/firefox/addon/tampermonkey/)

#### Шаг 2. Установите скрипт автозаполнения
1. Нажмите на иконку установленного **Tampermonkey** в панели расширений вашего браузера (обычно вверху справа) и выберите **«Создать новый скрипт»** (Create a new script).
2. Полностью удалите весь шаблонный текст в открывшемся редакторе.
3. Откройте файл [мэш_soft_skills_autofill_pro.user.js](мэш_soft_skills_autofill_pro.user.js) в этом проекте, скопируйте его содержимое целиком (Ctrl+A -> Ctrl+C).
4. Вставьте скопированный код в редактор Tampermonkey (Ctrl+V).
5. Нажмите **Файл** (File) -> **Сохранить** (Save) в верхнем меню редактора.

### 🎮 Как пользоваться:
1. Перейдите на сайт [school.mos.ru](https://school.mos.ru) и войдите в свой личный кабинет учителя.
2. Перейдите в журнал любого класса. В правом нижнем углу экрана появится круглая синяя кнопка **`⚡`**. Нажмите на неё, чтобы открыть панель управления.
3. **Вкладка «Запуск»**:
   * Выберите группу (класс) для заполнения (скрипт автоматически определит класс, если вы уже находитесь на его странице, либо позволит выбрать из списка).
   * Выберите нужный триместр.
   * Выберите режим: «Только новые» или «Исправить/Перезаписать».
   * Нажмите **«🔍 Анализировать класс и оценки»**.
4. **Вкладка «Превью»**:
   * Перед вами откроется список учеников с их средними баллами.
   * Оценки по критериям (Самоорганизация, Самообразование, Саморегуляция, Коммуникация) проставятся автоматически.
   * Если вы хотите изменить оценку конкретного ученика, выберите другое значение в выпадающем списке прямо напротив его имени.
   * Снимите галочки с тех учеников, кого заполнять не нужно.
   * Нажмите **«🚀 Отправить данные в МЭШ»**.
5. Следите за процессом во вкладке **«Консоль»**. По окончании вы увидите отчет.
6. **Вкладка «Настройки»**:
   * Позволяет регулировать пороги среднего балла для оценок.
   * Показывает автоматически захваченный **Bearer Token** и **Profile ID**. Если у вас возникла ошибка авторизации, просто обновите страницу электронного журнала, чтобы скрипт перехватил свежую сессию, или введите параметры вручную.

## 🐍 Способ 2. Запуск через Python (Для продвинутых пользователей)
*Если вы предпочитаете запускать скрипты через консоль на своем компьютере.*

### 🔧 Требования
1. **Установите Python**:
   * **Windows**: Скачайте с официального сайта. При установке обязательно отметьте галочку **«Add Python to PATH»**.
   * **macOS**: Установите через Homebrew или скачайте с официального сайта.
2. **Установите библиотеку requests**:
   ```bash
   pip install requests
   ```

### 🚀 Инструкция:
1. Откройте **school.mos.ru** в браузере Chrome и войдите в систему.
2. Нажмите **F12** (или ПКМ -> «Исследовать») -> вкладка **Network** (Сеть).
3. Обновите страницу журнала.
4. Кликните на любой запрос к `school.mos.ru/api/...` в списке.
5. В разделе **Headers** -> **Request Headers** найдите и скопируйте:
   * Значение заголовка `Authorization` (без слова `Bearer `).
   * Значение заголовка `Profile-Id`.
6. Запустите скрипт:
   * **Обычный запуск (для нового заполнения)**:
     ```bash
     python3 mesh_soft_skills_autofill_final.py
     ```
   * **Запуск исправления (если нужно удалить старые и заполнить заново по триместрам)**:
     ```bash
     python3 mesh_soft_skills_fix.py
     ```
7. Введите токен и Profile-Id в консоли и следуйте текстовому меню.
