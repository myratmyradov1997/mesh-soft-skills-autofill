# МЭШ — Автозаполнение Soft Skills

Скрипт автоматически заполняет оценку «soft skills» для всех учеников
на основе их среднего балла по предмету.

---

## 📋 Что делает скрипт

1. Получает список всех ваших групп (классов) из МЭШ
2. Для каждого ученика загружает его отметки за учебный год
3. Вычисляет средний балл
4. На основе среднего балла определяет ответы на вопросы soft skills
5. Отправляет ответы в МЭШ

**Вариативность ответов** — ответы не одинаковые для всех вопросов:
- «Выступления перед аудиторией» и «Интерес вне программы» — чаще ниже
- «Вежливость», «Домашние задания» — чаще выше
- У каждого ученика свой уникальный профиль ответов

## 🔧 Как установить

### 1. Установи Python

**macOS:**
```bash
# Проверь, установлен ли Python:
python3 --version
```

**Windows:**
- Скачай Python с https://www.python.org/downloads/
- При установке ОБЯЗАТЕЛЬНО отметь галочку **«Add Python to PATH»**

### 2. Установи библиотеку requests

Открой **Терминал** (macOS) или **Командную строку** (Windows) и выполни:

```bash
pip install requests
```

или если не работает:

```bash
pip3 install requests
```

## 🚀 Как запустить

### Шаг 1. Скопируй данные из браузера

1. Открой **school.mos.ru** в браузере Chrome, войди в свой аккаунт
2. Нажми **F12** (или ПКМ → «Исследовать») — откроются инструменты разработчика
3. Перейди на вкладку **Network** (Сеть)
4. **Обнови страницу** (F5)
5. Кликни на **любой** запрос в списке (например, тот что начинается с `api/ej/...`)
6. В правой панели → вкладка **Headers** → раздел **Request Headers**
7. Скопируй **два** значения:
   - **`Authorization: Bearer eyJ...`** — весь длинный текст после «Bearer »
   - **`Profile-Id: 17905177`** — число из заголовка Profile-Id (у вас будет своё)

![Где искать токен и Profile-Id](https://i.imgur.com/placeholder.png)

### Шаг 2. Запусти скрипт

```bash
python3 mesh_soft_skills_autofill.py
```

### Шаг 3. Вставь данные

1. Вставь **Bearer токен** (Cmd+V / Ctrl+V)
2. Вставь **Profile-Id** (число)
3. Выбери группы для заполнения
4. Жди завершения

## ⚙️ Настройка порогов

В файле `mesh_soft_skills_autofill.py` можно изменить пороги
в разделе **НАСТРОЙКИ** (строки ~25-45):

```python
# Пример: сделать требования строже
THRESHOLDS = {
    "self_org": [4.8, 4.0, 3.0],  # было [4.6, 3.7, 3.0]
    "self_edu": [4.9, 4.5, 3.5],  # было [4.8, 4.0, 3.3]
    ...
}
```

| Параметр | Что значит |
|----------|-----------|
| `[4.6, 3.7, 3.0]` | ≥4.6 → «Всегда», ≥3.7 → «Часто», ≥3.0 → «Редко», <3.0 → «Никогда» |
| `QUESTION_OFFSETS` | Смещения для вариативности ответов |

## ❓ Частые вопросы

**Где взять токен?**
F12 → Network → обнови страницу → кликни любой запрос к api → 
заголовок Authorization. Токен действует ~7 дней.

**Сколько это занимает времени?**
~1 секунда на ученика. На класс из 30 человек — ~30 секунд.

**Можно отменить?**
Нажми Ctrl+C в терминале. Уже отправленные ответы останутся.

**Ошибка «Токен истёк»?**
Обнови страницу в браузере, скопируй новый токен и запусти снова.
Уже заполненные ученики будут пропущены.

**Надо ли что-то менять в коде?**
Нет. Просто запусти и следуй инструкциям.

**Можно запустить на Windows?**
Да. Нужен Python (скачать с python.org) и библиотека requests.
